# Skill-SD

**Skill-SD** is a skill search and discovery interface — a single-page web application for browsing and finding AI agent skills.

## Contents

| File | Description |
|---|---|
| `index.html` | Main skill discovery interface |
| `style.css` | Styling for the interface |
| `favicon.svg` | Site icon |
| `assets/` | Supporting assets |

## Usage

Open `index.html` in a browser to browse and search the skill index.

```bash
# Open locally
start skills/Skill-SD/index.html   # Windows
open skills/Skill-SD/index.html    # macOS
```
